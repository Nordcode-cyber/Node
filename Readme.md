###                                               Node.js

Entorno de ejecución de Javascript, podemos ejecutar código de Javascript fuera del navegador, en la linea de comandos,
de la terminal de nuestro sistema operativo. Este entorno es orientado a eventos asíncronos, un evento asíncrono se ejecuta, independientemente del proceso principal de la aplicacion.Node.js es usado para,
desarrollo Back-end, desarrollar APIs, además cuenta con un amplio ecosistema de biblioteca y módulos que facilitan el desarrollo de aplicaciones web y de servidor.
