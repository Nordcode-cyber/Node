###                                               Node.js

Entorno de ejecución de Javascript, podemos ejecutar código de Javascript fuera del navegador, en la linea de comandos,
de la terminal de nuestro sistema operativo. Este entorno es orientado a eventos asíncronos, un evento asíncrono se ejecuta, independientemente del proceso principal de la aplicacion.Node.js es usado para,
desarrollo Back-end, desarrollar APIs, además cuenta con un amplio ecosistema de biblioteca y módulos que facilitan el desarrollo de aplicaciones web y de servidor.

####                                          Evento síncrono
Evento que se ejecuta como parte del proceso principal de la aplicación.

####                                         Servidor http

Node JS permite crear servidores web de forma sencilla, para manejar solicitudes y respuestas web.
El módulo ```http``` es un módulo principal de ```Node.js``` (un módulo incluido en la fuente de Node.js,
que no requiere la instalación de recursos adicionales). El módulo ```http``` proporciona la
funcionalidad para crear un servidor HTTP utilizando el método ```http.createServer()``` . Para crear la
aplicación, cree un archivo que contenga el código JavaScript.
